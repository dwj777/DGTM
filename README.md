# DGTM Regulatory Network Model

This repository contains the Python code used for parameter selection of the DGTM regulatory network model described in our manuscript.

## Requirements

numpy

scipy

matplotlib

pandas

## Running

Run

```python
DGTM.py
```

The code performs

- parameter sampling
-  Equilibrium searching
-  Stability assessment
- Trajectory convergence verification
- Pattern matching
